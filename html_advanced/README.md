# HTML Advanced

## Project Overview

This is the HTML Advanced implementation of the SmileSchool webpage, created as part of the ALCHE Front-End Web Development Project. The project focuses on building a semantic HTML5 structure based on a Figma design specification.

## Description

SmileSchool is an educational platform where users can learn various smiling techniques from professional instructors. The webpage features tutorial videos, testimonials, membership information, and FAQ sections.

## Project Structure

```
html_advanced/
└── index.html       # Main HTML file with semantic structure
```

## Features

### HTML Structure Includes:

1. **Header Section**
   - Logo with clickable link
   - Navigation menu with 3 links (COURSES, PRICING, LOGIN)

2. **Banner Section**
   - Hero section with title, subtitle, and CTA button
   - "Learn from the pros" section featuring 4 instructor profiles

3. **Quote Section**
   - Testimonial with avatar image
   - Blockquote with author details

4. **Videos Section**
   - Section title
   - 4 video tutorial cards with:
     - Thumbnail images
     - Video title and description
     - Author information
     - Star rating and duration

5. **Membership Section**
   - Section title
   - 4 membership benefit items with icons
   - CTA button

6. **FAQ Section**
   - Section title
   - 2x2 grid of question-answer pairs

7. **Footer Section**
   - Logo
   - Social media icons (Facebook, Twitter, Instagram)
   - Copyright notice

## Technical Specifications

- **HTML Version:** HTML5
- **Validation:** W3C Valid HTML
- **Semantics:** Uses semantic HTML5 tags (`<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`, `<blockquote>`)
- **Accessibility:** Includes proper `alt` attributes for all images
- **Best Practices:** Clean indentation and proper tag nesting

## Setup Instructions

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. Note: This is HTML-only. Styling will be added in the CSS Advanced task.
